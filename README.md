# ValuationGraph
Depicting shares for startup company (Exit Valuation vs Price per Share) using nivo library.

I have plotted Exit valuation and price per share for stakeholders using ReactJS with nivo libarary. 

Steps to run:
step1: npm install
step2: npm start
